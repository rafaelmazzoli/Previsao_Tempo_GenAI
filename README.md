# Previsao Tempo Prompt
Aplicativo de prompt em Python para pesquisar informações sobre a previsão do tempo de qualquer cidade do mundo.

Utilizamos as APIs gratuitas da [OpenWeatherMap](https://openweathermap.org/api).

## Para usar
Passo-a-Passo:
- Ter o Python instalado na máquina
- Baixar o projeto localmente
- Instalar as dependências do projeto
- Criar uma conta na [OpenWeatherMap](https://openweathermap.org/register)
- Criar um arquivo ```.env``` contendo a [API Key](https://home.openweathermap.org/api_keys) da sua conta OpenWeatherMap contendo o seguinte:
  - ```API_KEY_OPEN_WEATHER_SERVICE="SUA_API_KEY"```
- Executar o programa pelo arquivo ```main.py```
